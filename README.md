# Cайт портфолио

Макет сделан мной. Дизайн (PSD) и верстка (HTML, LESS, jQuery). Сделан для курса обучения верстки сайтов в WebCademy.ru

## Что внутри:

* Оптимизация под декстоп, смартфоны, планшеты.
 * Навигация по странице, плавный page scroll
  * Мобильная навигация
   * Фильтр карточек в Портфолио
    * Форма обратной связи (Ajax, PHP)    

[Онлайн демо](https://igor94ua.github.io/git/index.html)
